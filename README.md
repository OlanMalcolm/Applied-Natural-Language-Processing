This repository contains all coursework, assignments, and projects completed as part of CSE 8803: Applied Natural Language Processing, an advanced elective in the Master of Science in Analytics program at Georgia Tech.

The course focuses on practical and theoretical approaches to processing, analyzing, and modeling natural language data. Topics include text preprocessing, feature extraction, word embeddings, language modeling, sequence labeling, topic modeling, sentiment analysis, and modern neural network architectures for NLP, with an emphasis on real-world applications.
